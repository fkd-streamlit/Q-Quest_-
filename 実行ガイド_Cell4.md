# Cell 4 実行ガイド

## 実行方法

### 方法1: Jupyter Notebookから実行（推奨）

1. **Jupyter Notebookを起動**
   ```bash
   jupyter notebook Q_QUEST_量子神託.ipynb
   ```

2. **Cell 4を実行**
   - Cell 4をクリックして選択
   - **Shift + Enter** で実行
   - プロンプトが表示されたら、願いや問いを入力
     - 例: 「世界平和に貢献できる人間になる」
   - 3D球体が表示されます

### 方法2: Pythonスクリプトとして実行

1. **テスト用スクリプトを実行**
   ```bash
   python test_cell4.py
   ```

2. **出力**
   - 3D球体の視覚化が表示されます
   - コンソールにキーワードと格言が表示されます

## 実行時に必要なパッケージ

```bash
pip install numpy matplotlib
```

## 期待される出力

実行すると以下が表示されます：

1. **コンソール出力**:
   - 入力された言葉
   - 抽出されたキーワード

2. **3D球体の可視化**:
   - 黒い背景の3D球体
   - 金色の中心語（入力したキーワード）
   - 白色の他の単語が球面上に配置
   - 青/赤の線で繋がれたネットワーク

3. **格言**:
   - キーワードに基づいた適切な格言が表示

## トラブルシューティング

### 3D表示が表示されない場合

- `plt.show()` が正しく動作するか確認
- バックエンドを確認: `matplotlib.get_backend()`
- インタラクティブモードを使用: `plt.ion()`

### 日本語が表示されない場合

- 日本語フォントをインストール
- matplotlibの設定を確認

### エラーが出る場合

パッケージのバージョンを確認：
```bash
pip install --upgrade numpy matplotlib
```
