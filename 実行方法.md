# Q-Quest 量子神託 実行方法

## 方法1: バッチファイルで実行（最も簡単）⭐推奨

### Windowsの場合

1. **エクスプローラーでプロジェクトフォルダを開く**
   ```
   C:\Users\FMV\Desktop\Q-Quest_量子神託
   ```

2. **`start_app.bat` をダブルクリック**

3. **ブラウザが自動的に開きます**
   - 通常は `http://localhost:8501` で開きます
   - 開かない場合は、コマンドプロンプトに表示されるURLをコピーしてブラウザに貼り付け

4. **アプリを停止する場合**
   - コマンドプロンプトウィンドウで `Ctrl+C` を押す

---

## 方法2: コマンドプロンプトで実行

### 手順

1. **コマンドプロンプト（cmd）を開く**

2. **プロジェクトフォルダに移動**
   ```bash
   cd C:\Users\FMV\Desktop\Q-Quest_量子神託
   ```

3. **依存パッケージをインストール（初回のみ）**
   ```bash
   pip install -r requirements.txt
   ```

4. **Streamlitアプリを起動**
   ```bash
   streamlit run app.py
   ```

5. **ブラウザが自動的に開きます**
   - 開かない場合は、コマンドプロンプトに表示されるURL（例: `http://localhost:8501`）をコピーしてブラウザに貼り付け

1. **アプリを起動後、サイドバーの「📊 設定ファイル」セクションでExcelファイルをアップロード**

2. **「✅ 設定ファイルを読み込みました」と表示されれば成功**

## トラブルシューティング

### エラー: "ModuleNotFoundError: No module named 'streamlit'"

**解決方法:**
```bash
pip install -r requirements.txt
```

### エラー: "ModuleNotFoundError: No module named 'pandas'"

**解決方法:**
```bash
pip install pandas openpyxl
```

### ポート8501が既に使用されている

**解決方法:**
- 他のStreamlitアプリが起動している場合は、それを停止する
- または、別のポートを使用:
  ```bash
  streamlit run app.py --server.port 8502
  ```

### ブラウザが自動的に開かない

**解決方法:**
- コマンドプロンプトに表示されるURL（例: `http://localhost:8501`）をコピー
- ブラウザのアドレスバーに貼り付けてEnter

## アプリの使い方

### 1. 基本デモ
- QUBOモデルを使用した基本的な「縁」のデモンストレーション
- 「実行」ボタンをクリック

### 2. 対話型量子神託
- 悩みや気持ちを入力すると、パーソナライズされた「縁」を提示
- テキストエリアに願い事を入力して「神託を求める」ボタンをクリック

### 3. 言葉のエネルギー球体視覚化
- 入力した言葉からキーワードを抽出し、3D球体で可視化

### 4. 絵馬納め
- 願いを絵馬に書いて納めると、12神のうち適切な神が現れて神託を授ける
