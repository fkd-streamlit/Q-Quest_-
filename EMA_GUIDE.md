# 🎋 絵馬納め機能の使い方

## 概要

「絵馬納め」機能では、あなたの願いを絵馬に書いて納めることで、**秋葉三尺坊大権現**が現れて神託を授けてくれます。

## 使い方

### 1. モード選択

サイドバーから「**絵馬納め**」を選択してください。

### 2. 願いを書く

絵馬に願いを書きます：
- 健康に関する願い
- 仕事や学業に関する願い
- 家族や友人に関する願い
- その他、あなたの心に浮かぶ願い

**例**:
```
健康で過ごせますように
仕事がうまくいきますように
家族が幸せでありますように
```

### 3. 絵馬を納める

「🎋 絵馬を納める」ボタンをクリックすると：
1. 絵馬が納められる演出が表示されます
2. 秋葉三尺坊大権現が現れるアニメーションが始まります
3. キャラクターが浮遊し、光る効果が表示されます
4. 神託が授けられます

### 4. 神託を受け取る

秋葉三尺坊大権現から以下の情報が授けられます：
- **選ばれた縁**: QUBOアルゴリズムで選ばれた格言
- **ことば**: 季節の言葉と格言を組み合わせた詩
- **次の一歩**: あなたの心の状態に基づいたアドバイス
- **心の傾き**: 5つの感情次元の分析結果

## アニメーション効果

### 実装されているアニメーション

1. **フェードイン**: キャラクターがゆっくりと現れます
2. **浮遊**: キャラクターが上下に浮遊します
3. **光る効果**: タイトルが金色に光ります
4. **キラキラ**: 背景に星がキラキラと輝きます
5. **シマー効果**: 背景が光の帯のように流れます

### 技術的な詳細

- **HTML/CSS/JavaScript**を使用してアニメーションを実装
- **CSS Keyframes**でアニメーション効果を定義
- **Streamlit Components**でHTMLを埋め込み

## 秋葉三尺坊大権現について

秋葉三尺坊大権現（あきばさんじゃくぼうだいごんげん）は、日本の仏教・神道における火防（ひぶせ）の神様です。

- **特徴**: 白い狐に乗った姿で描かれることが多い
- **ご利益**: 火災除け、家内安全、商売繁盛
- **信仰**: 静岡県の秋葉山本宮秋葉神社などで信仰されている

## カスタマイズ

### キャラクターの見た目を変更する

`app.py`の`character_html`変数内で以下を変更できます：

- **キャラクター絵文字**: `.character`クラスの内容（現在は`🦊✨`）
- **タイトル**: `.title`クラスの内容
- **色**: `color`プロパティを変更
- **サイズ**: `font-size`を変更
- **アニメーション速度**: `animation`の時間を変更

### アニメーション効果を追加する

CSSの`@keyframes`を追加して、新しいアニメーション効果を定義できます。

**例**: 回転アニメーションを追加
```css
@keyframes rotate {
    from { transform: rotate(0deg); }
    to { transform: rotate(360deg); }
}
```

## トラブルシューティング

### アニメーションが表示されない場合

1. ブラウザのJavaScriptが有効になっているか確認
2. ブラウザのキャッシュをクリア
3. 別のブラウザで試す

### 文字化けする場合

- ブラウザのフォント設定を確認
- 日本語フォントがインストールされているか確認
