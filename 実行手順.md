# Q-Quest 量子神託 実行手順

## 方法1: Jupyter Notebookで実行（推奨）

### 1. 依存パッケージのインストール

コマンドプロンプト（cmd）またはPowerShellで、プロジェクトディレクトリに移動して以下を実行：

```bash
cd C:\Users\FMV\Desktop\Q-Quest_量子神託
pip install -r requirements.txt
```

### 2. Jupyter Notebookの起動

```bash
jupyter notebook Q_QUEST_量子神託.ipynb
```

ブラウザが自動的に開き、Notebookが表示されます。

### 3. 実行方法

#### Cell 0を実行（基本デモ）

1. Cell 0をクリックして選択
2. **Shift + Enter** キーを押して実行
3. 以下が表示されます：
   - 低エネルギー上位の候補リスト
   - エネルギー地形のグラフ
   - 縁のネットワーク図
   - 量子おみくじの結果

#### Cell 2を実行（対話型量子神託）

1. Cell 2をクリックして選択
2. **Shift + Enter** キーを押して実行
3. プロンプトが表示されたら、悩みや気持ちを入力（例：「疲れていて決断ができない」「眠い。休みたい。」）
4. 以下が表示されます：
   - 推定された心の傾き（5つの次元）
   - 低エネルギー候補Top3
   - パーソナライズされた量子おみくじ
   - エネルギー地形のグラフ

## 方法2: VS Codeから実行

1. VS Codeで `Q_QUEST_量子神託.ipynb` を開く
2. 各Cellの上部にある「実行」ボタンをクリック
3. または、Cellを選択して **Shift + Enter** を押す

## 方法3: Google Colabで実行

1. Google Colab（https://colab.research.google.com/）にアクセス
2. `Q_QUEST_量子神託.ipynb` をアップロード
3. Cell 0またはCell 2を実行

**注意**: Cell 2には `!apt-get -y install fonts-ipaexfont` コマンドが含まれており、これはColab環境用です。ローカル環境では必要ありません（警告が出る場合がありますが、実行には影響しません）。

## 方法4: Cell 4の実行（QUBO × 量子神託 UI）

1. Cell 4を選択してShift+Enterで実行
2. プロンプトで願いや問いを入力
   - 例：「世界平和に貢献できる人間になる」
3. 3D球体が表示され、キーワードが金色で強調表示されます
4. 関連する言葉が線で繋がれ、画面下部に格言が表示されます

**詳細**: `実行ガイド_Cell4.md` を参照してください。

## トラブルシューティング

### Jupyter Notebookが起動しない場合

```bash
# Jupyter Notebookをインストール
pip install jupyter

# または、JupyterLabを使用
pip install jupyterlab
jupyter lab Q_QUEST_量子神託.ipynb
```

### パッケージが見つからないエラー

```bash
pip install numpy matplotlib jupyter ipykernel
```

### 日本語フォントの警告

ローカル環境では日本語フォントの警告が出る場合がありますが、実行には影響しません。日本語を正しく表示したい場合は、日本語フォントをインストールしてください。
