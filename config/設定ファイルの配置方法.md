# 設定ファイル（Excel）の配置方法

## 5つのExcel設定ファイル

1. **akiba12_character_list.xlsx** - 12神の基本情報
2. **格言.xlsx** - 格言データ（オプション）
3. **akiba12_character_to_vow_K.xlsx** - k行列（12×12）
4. **akiba12_character_to_axis_L.xlsx** - l行列（12×4）
5. **sense_to_vow_initial_filled_from_user.xlsx** - sense_to_vow行列（8×12）

## 配置場所の選択肢

### オプション1: ローカル開発用（推奨）

**配置場所**: `config/` フォルダー

```
Q-Quest_量子神託/
└── config/
    ├── akiba12_character_list.xlsx
    ├── 格言.xlsx
    ├── akiba12_character_to_vow_K.xlsx
    ├── akiba12_character_to_axis_L.xlsx
    └── sense_to_vow_initial_filled_from_user.xlsx
```

**メリット**:
- ローカル開発時に簡単にアクセスできる
- ファイルパスが明確

**注意**:
- **GitHubには含めません**（`.gitignore`で除外）
- 個人用設定のため、リポジトリには含めない

### オプション2: Streamlit Cloud（公開用）

**配置場所**: アプリからアップロード

Streamlit Cloudで公開する場合、ExcelファイルはGitHubに含めません。代わりに：

1. Streamlitアプリのサイドバーから「5つのファイル（推奨）」を選択
2. 各Excelファイルをアップロード
3. ファイルはセッションごとに保持されます

**メリット**:
- 個人用設定をGitHubに含めない
- ユーザーごとに異なる設定を使用可能

### オプション3: テンプレートファイル（オプション）

**配置場所**: `config/` フォルダー（テンプレートとして）

```
Q-Quest_量子神託/
└── config/
    ├── akiba12_character_list.example.xlsx
    ├── 格言.example.xlsx
    ├── akiba12_character_to_vow_K.example.xlsx
    ├── akiba12_character_to_axis_L.example.xlsx
    └── sense_to_vow_initial_filled_from_user.example.xlsx
```

**メリット**:
- GitHubに含めることができる（テンプレートとして）
- 他のユーザーが設定ファイルの構造を理解しやすい

**注意**:
- `.example.xlsx` という拡張子を使用
- `.gitignore`で通常の`.xlsx`は除外、`.example.xlsx`は含める

## 推奨構成

### ローカル開発環境

```
Q-Quest_量子神託/
├── config/
│   ├── README.md                    # このファイル
│   ├── akiba12_character_list.xlsx  # あなたの設定ファイル
│   ├── 格言.xlsx
│   ├── akiba12_character_to_vow_K.xlsx
│   ├── akiba12_character_to_axis_L.xlsx
│   └── sense_to_vow_initial_filled_from_user.xlsx
```

### GitHubリポジトリ

```
Q-Quest_量子神託/
├── config/
│   └── README.md                    # 設定ファイルの説明のみ
```

Excelファイルは含めません（`.gitignore`で除外）

### Streamlit Cloud

- アプリのサイドバーからExcelファイルをアップロード
- デフォルト設定も使用可能

## 使用方法

### ローカル開発

1. `config/` フォルダーにExcelファイルを配置
2. Streamlitアプリを起動
3. サイドバーから「5つのファイル（推奨）」を選択
4. `config/` フォルダーからファイルを選択してアップロード

または、アプリを修正して `config/` フォルダーから自動的に読み込むようにすることも可能です。

### Streamlit Cloud

1. アプリのサイドバーから「5つのファイル（推奨）」を選択
2. 各Excelファイルをアップロード
3. 設定が読み込まれます

## 自動読み込み機能（オプション）

アプリを修正して、`config/` フォルダーから自動的にExcelファイルを読み込むことも可能です。この場合：

- ローカル開発: `config/` フォルダーから自動読み込み
- Streamlit Cloud: アップロード機能を使用
