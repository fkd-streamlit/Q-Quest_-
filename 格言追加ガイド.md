# キャラクター毎の格言追加ガイド

## 概要

Q-Quest 量子神託アプリでは、各キャラクター（12神）に個別の格言を設定できます。格言を追加することで、各キャラクターが現れた際に、そのキャラクターに特有の神託を表示できます。

## 格言を追加する理由

### ✅ 推奨される理由

1. **キャラクターの個性を表現**
   - 各キャラクターに特有の格言を設定することで、そのキャラクターの性格や役割を表現できます
   - ユーザーが「このキャラクターらしい」と感じられる体験を提供できます

2. **よりパーソナライズされた神託**
   - 選ばれたキャラクターに応じて、適切な格言を表示できます
   - ユーザーの願い事や心の状態に応じた、より具体的なアドバイスを提供できます

3. **一貫性のある体験**
   - 同じキャラクターが選ばれた場合、常に同じ格言が表示されるため、一貫性のある体験を提供できます

## Excelファイルへの格言追加方法

### 方法1: 12神基本情報ファイルに追加（推奨）

`akiba12_character_list.xlsx` ファイルの「格言」列に、各キャラクターの格言を追加します。

#### 複数格言にする（おすすめ）

「格言」セルには **複数の格言** を入れられます。アプリはユーザー入力からキーワードを抽出し、マッチしやすい格言を **上位2つ程度** 選んで提示します。

- **改行区切り**（一番簡単）:

```
成長::千里の道も一歩から
不安::境界（しきい）を越えるのは、静かな一歩でよい。
```

- **`||` 区切り**:

```
成長::千里の道も一歩から||不安::境界（しきい）を越えるのは、静かな一歩でよい。
```

- **タグなし**もOK（全文検索でゆるくマッチします）:

```
焦らず、一歩ずつ。答えは時間が教えてくれる。
```

**タグ形式**: `タグ1,タグ2::本文`  
例）`成長,挑戦::千里の道も一歩から`

**Excelファイルの構造：**

| ID | 名前 | 名前(英語) | 属性 | 絵文字 | 説明 | **格言** |
|----|------|-----------|------|--------|------|---------|
| 0 | スサノオ | Susanoo | 火 | 🔥 | 秋葉原の守護神... | **あなたの格言をここに** |
| 1 | 獅子 | Shishi | 電 | ⚡ | 秋葉原の原点... | **あなたの格言をここに** |
| ... | ... | ... | ... | ... | ... | ... |

### 方法2: 3つの別ファイルを使用している場合

`akiba12_character_list.xlsx` ファイルがない場合でも、k行列とl行列から自動的にキャラクター名を取得しますが、**格言は空になります**。

この場合、後で `akiba12_character_list.xlsx` ファイルを作成して、格言を追加することを推奨します。

## 格言の書き方のヒント

### 良い格言の例

1. **キャラクターの役割を反映**
   - 例：「接点と結合。協力=「チームワーク」の神。」→ 「つながりこそが力である。一人ではできないことも、協力すれば成し遂げられる。」

2. **ユーザーへの具体的なアドバイス**
   - 例：「冷静=「沈着冷静」の神。」→ 「焦らず、一歩ずつ。答えは時間が教えてくれる。」

3. **キャラクターの属性を表現**
   - 例：「火伏せ=「炎上回避」の神。」→ 「勢いを大切に、しかし冷静さも忘れずに。」

### 格言の長さ

- **推奨**: 20～50文字程度
- **短すぎる場合**: 意味が伝わりにくい
- **長すぎる場合**: 表示が読みにくくなる可能性

## 現在の動作

### 格言がある場合

- 選ばれたキャラクターの格言が神託として表示されます
- 格言の出典として、キャラクター名が表示されます

### 格言がない場合

- デフォルトの格言「今この瞬間を大切に。すべては縁で繋がっている。」が表示されます
- または、キャラクターの説明文が表示される場合があります

## 実装の詳細

アプリは以下の順序で格言を取得します：

1. **キャラクター変数から選ばれた神の格言**（階層構造の場合）
2. **選ばれた神の `maxim` フィールド**
3. **選ばれた神の `description` フィールド**（格言がない場合）
4. **デフォルトの格言**（上記すべてがない場合）

## 推奨事項

### ✅ 推奨される設定

- **すべてのキャラクターに格言を設定する**
- **各キャラクターの役割や属性を反映した格言を設定する**
- **ユーザーが「このキャラクターらしい」と感じられる格言を設定する**

### ⚠️ 注意点

- 格言がない場合でもアプリは動作しますが、より良い体験のためには格言を追加することを推奨します
- 格言は日本語で記述してください（英語でも動作しますが、UIは日本語です）
