## 🎯 活用シーン

### 1. 新メンバーのオンボーディング

**推奨読書順序:**
1. `PROJECT_VISION.md` - プロジェクトの全体像を理解
2. `CULTURAL_ELEMENTS.md` - 文化的背景を理解
3. `DEVELOPMENT_ROADMAP.md` - 開発計画を確認
4. `README_EN.md` - 英語での説明も確認

### 2. プレゼンテーション資料の作成

- **投資家向け**: `PROJECT_VISION.md` をベースに
- **技術者向け**: `DEVELOPMENT_ROADMAP.md` をベースに
- **国際的な発表**: `README_EN.md` と `CULTURAL_ELEMENTS.md` を活用

- **文化的要素の実装**: `CULTURAL_ELEMENTS.md` を参照

- **設計思想の確認**: `PROJECT_VISION.md` を参照

### 4. GitHubでの公開

- **README.md**: プロジェクトのメイン説明（日本語）
